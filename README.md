# visitzuidlimburg-scraper

Scrapet hotel-detailpagina's uit de sitemap van visitzuidlimburg.nl en probeert per hotel de link te vinden achter
"Direct boeken" of "Naar de website". Output: CSV.

## Run (Codespaces)
1) Code → Codespaces → Create codespace
2) In terminal:
   pip install -r requirements.txt
   python main.py

## Output
- visitzuidlimburg_hotels_direct_links.csv
